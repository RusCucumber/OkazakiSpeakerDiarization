# Research Project Template

This repository is a research project template using R and Python.

## Directory Structure

```raw
project_name/
│
├── data/
│   ├── raw/                   # Raw data files
│   ├── processed/             # Processed data files
│   └── external/              # Data from external sources
│
├── R/
│   ├── scripts/               # R scripts for analysis
│   └── quarto_reports/        # Quarto documents for R analysis
│
├── python/
│   ├── scripts/               # Python scripts for analysis
│   ├── modules/               # Python modules used other Python programs
│   └── jupyter_notebooks/     # Jupyter Notebooks for Python analysis
│
├── results/
│   ├── figures/               # Generated figures and plots
│   ├── tables/                # Generated tables
│   └── models/                # Saved models
│
├── reports/
│   ├── interim/               # Interim reports
│   └── final/                 # Final reports
│
├── environment/
│   ├── R/                     # R environment files (e.g., renv, .Rprofile)
│   └── python/                # Python environment files (e.g., requirements.txt, environment.yml)
│
├── README.md                  # Project overview and setup instructions
└── .gitignore                 # Git ignore file
```
